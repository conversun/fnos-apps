# Fileview for fnOS

BaseMetas Fileview 是新一代通用型在线文件预览引擎，支持 Office、PDF、OFD、CAD、图片、代码文件、流程图、思维导图等数百种格式。

本 fnOS 包使用官方 Docker 镜像 `basemetas/fileview`，默认将 Web 服务映射到 `9000` 端口。

## Local Build

```bash
cd apps/fileview && ./update_fileview.sh
```
